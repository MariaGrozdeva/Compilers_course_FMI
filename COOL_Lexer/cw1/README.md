# COOL Lexer

COOL lexer project with an ANTLR4 lexer and a small C++ driver, plus helper scripts to build, clean, and run the tests.

## Project layout

```
cw1/
├─ src/                  # source files
│  ├─ CoolLexer.g4       # ANTLR4 lexer grammar
│  └─ drivers/
│     └─ LexerDriver.cpp # C++ driver
├─ gen/                  # auto-generated by the build
│  ├─ CoolLexer.h
│  └─ CoolLexer.cpp
├─ build/                # CMake build tree (artifacts)
│  └─ lexer              # executable produced by the build
├─ tests/               
│  └─ lexer/             # input/output test cases
└─ tools/                # helper scripts
   ├─ build.sh           # configure + compile
   ├─ clean.sh           # remove build & generated files
   └─ grade-lexer.sh     # run the full test suite
```

## Run a single test (from `cw1`)

```bash
./build/lexer < tests/lexer/086.strings.in
```

This feeds one test input to `build/lexer` and prints the token stream.

## Run all tests

```bash
./tools/grade-lexer.sh
```
